# 切向量化工具 Metatorio

切向量化是切向量正在锐意制作中的量化工具，主要用于异星工厂中的问题计算。切向量化使用了顶针的修辞手法，令人忍俊不禁。

# 目前实现的功能

- 加载异星工厂游戏数据
- 手动添加游戏机制
- 手动添加生产目标
- 求解出每个游戏机制的最优生产速率

# 计划实现的功能

- 警惕 XY Problem
- Steam 交互
- 保存和加载
- 更全面的信息展示
- 复原更多游戏机制：采矿、插件、研究……
- 带品质系统的计算
- 带地点限制的计算
- 带温度限制的计算
- 限制科技水平的自动配方选择

# 使用方法

## 添加游戏上下文
1. 打开程序后，点击左侧面板中的「异星工厂」选项。
2. 选择异星工厂可执行文件的路径
3. （可选）选择模组文件夹的路径
4. 点击「加载游戏上下文」按钮
5. 如果已经加载过一次，可以点击「加载缓存上下文」按钮跳过游戏导出过程
6. 加载完成后点击左侧的子视图按钮

## 工厂规划

1. 进入子视图后左上角可以添加各种物品的生产目标
2. 左下角可以添加游戏机制。
  - 「配方」是异星工厂最常见的生产方式，在组装机内进行物品转换
  - 「采矿」，从实体资源中采集原材料
3. 右侧面板显示当前的各个生产机制的分配比例
